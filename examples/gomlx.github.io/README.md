The examples here are used to as code snippets in the gomlx.github.io website. 

The are tagged with "//md:<tag>" trailling comments, that are used to mark where they are going to be used.

See the tool `github.com/gomlx/gomlx.github.io/cmd/sync_code` to synchronize code into the
`.md` files.

Since these examples are part of the test suite, we make sure that the code snippets don't drift in the docs
as the GoMLX API evolves.
